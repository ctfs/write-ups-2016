~ luvmarkdown ~

for all your markdown needs

got an important markdown file you need to share? then luvmarkdown is for you! we luv markdown just as much as you do. <3

we store each file at a unique url corresponding to the sha256 hash of the file. that way, you can upload your markdown knowing that it will never be tampered with. you can easily verify the integrity of a file by downloading it and ensuring that its filename is equal to hmac_sha256("luvmarkdown", file_contents).

credits to showdown and marked for the markdown conversion libraries.
